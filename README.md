# secux-repo

В этом репозитории хранятся дополнительные пакеты, собранные специально для дистрибутива Secux Linux.

## Установка и подключение

### 1. Редактирование pacman.conf

Откройте файл конфигурации `pacman` в любом удобном текстовом редакторе с правами суперпользователя (root). Например, используя nano:

```bash
sudo nano /etc/pacman.conf
```

### 2. Добавление репозитория

Прокрутите файл в самый низ и добавьте следующий блок строк:

```ini
[kolbanidze]
Server = https://kolbanidze.github.io/secux-repo/x86_64/
```

Сохраните изменения и закройте файл (в nano это делается нажатием Ctrl+X, затем Y и Enter).

### 3. Импорт GPG-ключей

Для того чтобы pacman доверял пакетам из этого источника, необходимо добавить публичный ключ в связку ключей вашей системы. Выполните следующую команду:

```bash
pacman-key --recv-keys CE48F2CC9BE03B4EFAB02343AA0A42D146D35FCE
```

После импорта рекомендуется локально подписать ключ, если настройки вашей системы требуют высокого уровня доверия (обычно это делается командой `pacman-key --lsign-key CE48F2CC9BE03B4EFAB02343AA0A42D146D35FCE`), однако в большинстве случаев достаточно простого импорта.

### 4. Обновление баз данных

Обновите списки пакетов, чтобы система увидела новый репозиторий:

```bash
sudo pacman -Syu
```

Теперь репозиторий подключен, и вы можете устанавливать пакеты стандартным способом через `pacman -S имя_пакета`.

## Доступные пакеты

В данный момент в репозитории поддерживаются следующие пакеты:

*   **shim-signed**
    Портированная версия загрузчика shim из Ubuntu. Подписан ключами microsoft. 

*   **secux-hooks**
    Набор специфических хуков для Secux Linux. Они автоматически выполняют необходимые системные процедуры при обновлении или установке пакетов.

*   **cryptopro-csp-k1**
    Пакет криптопровайдера CryptoPro CSP.
